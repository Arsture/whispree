자주 쓰는 전문 용어·고유명사를 등록해 두면 받아쓰기 정확도가 올라갑니다. 그리고 잘못 받아쓴 단어는 **Quick Fix** 로 그 자리에서 고치고 사전에 바로 추가할 수 있습니다.

## 도메인 단어 세트

설정 → 단어 사전에서 주제별 단어 묶음을 관리합니다. 각 세트에는:

- **단어(words)** — STT 인식 힌트로 주입되고, 동시에 AI 교정의 용어집(glossary)으로 쓰입니다.
- **교정 매핑(mappings)** — `원래말 → 고칠말` 규칙. AI 교정 단계에서만 적용됩니다(STT에는 미적용).
- 세트별 켜기/끄기 토글과 단어/매핑 개수.

**기본 세트**도 한 번에 추가할 수 있습니다: “IT/개발”(개발 용어 ~40개), “통계”(통계 용어 ~24개), 그리고 빈 “사용자 정의”.

:::note
어떤 STT 엔진이 단어 힌트를 받느냐는 엔진에 따라 다릅니다 — WhisperKit과 MLX Audio가 도메인 단어를 인식에 활용합니다. 매핑은 AI 교정이 켜져 있을 때 동작합니다.
:::

## Quick Fix — 즉석 교정 + 등록

오인식된 단어를 발견했을 때, 다시 받아쓸 필요 없이 바로 고칩니다.

1. 아무 앱에서나 고칠 텍스트를 선택합니다.
2. **`Ctrl+Shift+D`** 를 누릅니다. (기본 단축키)
3. Whispree가 선택 텍스트를 가져와 패널을 띄웁니다. 올바른 단어/매핑을 입력합니다.
4. **Enter** 로 저장하면, 대상 앱의 텍스트가 교체되고 동시에 사전에 등록됩니다. (Esc로 취소.)

두 가지 모드:

- **매핑 추가** *(기본 선택)* — `원래말 → 고칠말` 을 AI 교정 매핑으로 저장(STT에는 미적용).
- **단어 추가** — STT + AI 교정 사전 양쪽에 단어를 추가.

등록한 항목은 자동 생성되는 “Quick Fix” 단어 세트에 모입니다.

## 사전 동기화

설정 → 일반의 **사전 동기화** *(기본 꺼짐)* 를 켜면 단어 세트와 Quick Fix 단어를 **iCloud Drive** (또는 직접 지정한 JSON 경로 — Dropbox/Syncthing 등)로 동기화합니다. “지금 가져오기 / 지금 내보내기” 버튼으로 수동 동기화도 가능합니다. 여러 Mac에서 같은 사전을 공유할 때 유용합니다.
