title: 단축키

전역 단축키는 시스템 단축키보다 먼저 가로채집니다(CGEventTap). 재지정 가능한 단축키는 **설정 → 일반 → 단축키**에서 바꿀 수 있습니다.

## 전역 단축키

| 동작 | 기본 조합 | 재지정 |
| --- | --- | --- |
| **녹음 시작/정지** (토글 또는 눌러서 말하기) | `Ctrl+Shift+R` | 가능 |
| **Quick Fix** (선택 텍스트 즉석 교정) | `Ctrl+Shift+D` | 가능 |
| **스크린샷 첨부 토글** (녹음 중) | 왼쪽 `⌥` 약 0.5초 길게 | 불가(고정) |
| **현재 범위 취소** | `Esc` | 불가(고정) |

:::tip
단축키는 **수식 키 하나만**으로도 지정할 수 있습니다(예: 오른쪽 `⌥` 단독). 고른 조합이 알려진 시스템 단축키와 충돌하면 경고가 표시됩니다.
:::

## 화면 선택 패널

스크린샷 선택 패널이 떠 있을 때:

| 키 | 동작 |
| --- | --- |
| `↑` `↓` | 항목 이동 |
| `Space` | 선택/해제 |
| `Enter` | 확정 |
| `⌘Enter` | 미리보기(퀵룩) |
| `Esc` | 건너뛰기 |

## Quick Fix 패널

| 키 | 동작 |
| --- | --- |
| `Enter` | 저장 후 교체 |
| `Esc` | 취소 |

## 외부 자동화 (URL 스킴)

단축키 대신 URL로도 녹음을 제어할 수 있습니다 — Raycast, Stream Deck, Keyboard Maestro 등.

| URL | 동작 |
| --- | --- |
| `whispree://toggle` | 녹음 토글 |
| `whispree://start` / `whispree://push` | 녹음 시작 |
| `whispree://stop` / `whispree://release` | 녹음 종료 |
